# 3POO_5 — Porra Sistema

Txapelketa-porra baten kudeaketa-sistema. .NET 6.0 plataforman garatuta.

## Proiektuaren Deskribapena

Sistema honek erabiltzaileei txapelketa-porretan parte hartzeko aukera ematen die.
Parte-hartzaile bakoitzak irabazlea, bi finalistak eta goleatzailea aurreikus ditzake,
eta emaitzen arabera puntuak metatzen ditu.

## Puntuazio-Sistema

| Asmakizuna | Puntuak |
|---|---|
| Irabazlea asmatuta | +25 puntu |
| Bi finalistak asmatuta | +20 puntu |
| Gol-egile bat asmatuta | +3 puntu |

## Klaseen Egitura

- **[Porra](api/_3POO_5.Porra.html)** — Parte-hartzaile baten datuak eta puntuazio-logika.
- **[Program](api/_3POO_5.Program.html)** — Menu nagusia eta fitxategi-kudeaketa.

## Erabilera

```bash
# Konpilatu
dotnet build

# Exekutatu
dotnet run
```
